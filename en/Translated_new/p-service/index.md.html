The "Registration and Update" panel

Purpose: the "Registration and Update" panel is designed for:

  • Log in/log out to the account to activate/deactivate the panels included in the plugin (add-in) "3D Security Designer".

  • Receiving and installing updates.

  • Getting information about the "3D Security Designer" add-in (plugin).


Fig. 1. View of the "Registration and update" panel

Terminology

For an unambiguous understanding of the principles of operation and the user interface of the add-in, it is necessary to understand in what sense the following terms are used.

Account is a set of data about a user stored in a computer system, necessary for his identification (authentication) and providing access to his personal data and settings.

Registration of the “3D Security Designer” plugin (software) denotes the owner of the registered program and serves as a tool to protect his interests and the author's copyright.

Updating the "3D Security Designer" plugin (software) is a set of changes for software, fixes or improvements.

Family category is a group of elements used to model a construction object: fire alarms, devices for calling and alerting, windows, doors, walls, ceilings, etc.

Collision is a logical, geometric or physical contradiction (or ambiguity) in the user's instructions, making it impossible to implement these instructions. Before implementing the instructions, the elimination of collisions is required.

Commands of the "Registration and Update" panel

The Registration and Update panel provides commands to perform the following actions:

  • Login/logout to the account.
  • Updates to the add-in (plugin).
  • Getting information about the "3D Security Designer" add-in (plugin).
