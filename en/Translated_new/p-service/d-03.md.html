The "About the Program" command

Purpose: displaying information about the "3D Security Designer" version 1.0 program.


Fig. 1. Launching the "About the Program" command

To run the command, click the "About" button on the "Registration and Update" panel of the "3D Security Designer" tab. After that, the "About the program" window will be displayed (Fig. 2), which contains information about the name of the program, its version, developer and technical support.
