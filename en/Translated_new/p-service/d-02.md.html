The "Update" command

Purpose: update of the "3D Security Designer" plugin.


Fig. 1. Launching the Update command

The command is active only if the plugin developer has an update. If there are no updates, the "Update" command launch button will not be active (Fig. 2).


Fig. 2. View of the "Update" button when it is not active

To run the command, click the "Update" button on the "Registration and Update" panel of the "3D Security Designer" tab. After that, an Internet browser will be launched and a transition will be made to the page of the plugin developer's website, from which it will be possible to download the update in the form of an installation package (a file with the extension "msi").

To install the update, you need to close Autodesk Revit. After that, start the installation process (double-click on the downloaded file with the update) and follow the instructions of the installer.
