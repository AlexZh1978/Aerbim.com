The "Information on Element" command

Purpose: enable the user to quickly get information about the marking for the selected element. Output information: circuit name, mark format, mark value, alternative mark value.


Fig. 1. Launching the "Information on Element" command

To run the command, click the "Information on Element" button in the "Actions with Single Elements" group on the "Marking" panel of the 3D Security Designer tab. If no circuits have been created in the current document before, a window will be displayed informing the user about this.


Fig. 2. Warning about the absence of circuits

Otherwise, the status bar will display the request "Select element for which you want to get information about its marking". After the selection of the element (family) in the circuit, the "Information on Element Marking" window will be displayed.


Fig. 3. The "Information on Element Marking" window

The window control elements are responsible for the following functionality:

  1. The list of pairs "Parameter" - "Value" contains the values involved in the process of marking of the specified element (family). These parameters include: "Circuit Name", "Mark format", "Mark", "Alternative Mark".
  2. The "OK" button completes the "Information on Element" command and closes the "Information on Element Marking" window.

In case of a command failure, a corresponding message will be displayed, while information on element marking will not be displayed.


Fig. 4. Failure message
