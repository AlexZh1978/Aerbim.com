The "Alternative Mark" command

Purpose: enable the user to manually set an alternative mark value for an element (family) in the circuit. Alternative mark is not bound by the limitations of the format adopted for the circuit elements, and takes precedence when visualizing. The original mark of the element is saved and automatically becomes visible when the value of alternative mark is removed.


Fig. 1. Launching the "Alternative Mark" command

To run the command, click the "Alternative Mark" button in the "Actions with Single Elements" group on the "Marking" panel of the "3D Security Designer" tab. If no circuits have been created in the current document before, a window will be displayed informing the user about this


Fig. 2. Warning about the absence of circuits

Otherwise, the "Alternative Mark" window will be displayed.


Fig. 3. The "Alternative Mark" window

The window control elements are responsible for the following functionality:

  1. The text field "Alt. Mark Value" allows you to set a value for alternative mark of an element (family) in the circuit. The value cannot exceed more than ten characters (including space characters). The space characters are removed at the beginning and the end of the alternative mark line. An empty value for alternative mark means that the alternative mark will be removed.
  2. If the "For Single Element" checkbox is selected, the value of alternative mark will be set for only one element (family) in the circuit. Otherwise, the same value of the alternative mark will be used for several elements in the circuit.
  3. The "Apply" button starts the installation of alternative mark for the selected elements (families) in the circuits.
  4. The "Cancel" button cancels the "Alternative Mark" command and closes the "Alternative Mark" window.

After setting up the logic of the "Alternative Mark" command and clicking the "Apply" button, the setting of alternative mark values for the selected elements (families) in the circuits is started.

The status bar will prompt "Select element in the circuit for which you want to set alternative mark value". The ability to select multiple items depends on the value of the "For one element" checkbox.

The operation of the command:

  • If the "Alt. Mark Value" is not set, and the selected item doesn't have alternative mark, then nothing will happen.
  • If the "Alt. Mark Value" is not set, and the selected element has alternative mark, a warning about overwriting will be displayed.


Fig. 4. Warning about overwriting alternative mark

If the answer is "Yes", the value of alternative mark will be deleted. When the alternative mark is removed, the marking corresponding to the mark format for the circuit will be visualized. If the "Alt. Mark Value" is set, and the selected element doesn't have alternative mark, then it will be set. The value of alternative mark will be visualized automatically.

If the "Alt. Mark Value" is set, and the selected element has alternative mark, a warning about overwriting will be displayed.


Fig. 5. Warning about overwriting alternative mark

If the answer is "Yes", the value of alternative mark will be overwritten. The new value of alternative mark will be visualized automatically.
