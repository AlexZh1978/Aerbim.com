The "Information on Existing Circuits" window

Purpose: providing detailed information to the user about the marked circuits in the current document, about the circuits mark formats and the elements included in them.

The "Information on Existing Circuits" window can be called from the "Change Marks in Existing Circuit" window or from the "Delete Circuit and Marks" window by clicking on the "Information on Existing Circuits" button.

The window control elements are responsible for the following functionality:

  1. The "Circuit Name" list allows you to select one of the circuits previously created in the document to view information about it and its elements (families).
  2. The "Circuit Mark Format" field gives an idea of the mark format for the selected circuit. The field contains the format name and format structure separated by a colon.
  3. The "Element Marking Statistics" table contains statistical information about the circuit:
  • "Number of Elements" is the amount of elements in the circuit, including embedded ones.
  • "Counter Values in Mark Format" is the maximum amount of numbers provided by mark format.
  • "Unused Counter Values" is the amount of unused numbers that can be assigned to elements without violating the requirements of mark format.
  • "Out-of-Range Counter Values" is the amount of element numbers that lie outside the limits provided by mark format.
  1. The "Circuit Elements" table contains information about the elements (families) in the circuit selected for viewing, including the fields:
  • "Marking" — marking of an element in the circuit.
  • "Alternative marking" — alternative marking of an element in the circuit.
  • "Model" — the model of the device (as specified in the specifications).
  • "Name" — the full name of the equipment.
  • "Number of addresses" — the amount of addresses (for address devices).
  1. The "Apply" button sets the selected circuit as the current one in the parent window and closes the "Information on Existing Circuits" window.
  2. The "Cancel" button closes the "Information on Existing Circuits" window.
