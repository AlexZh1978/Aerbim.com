The "Manage Mark Formats" command

Purpose: enable the user to delete existing formats, export formats to a text file and import them from a text file.


Fig. 1. Launching the "Manage Mark Formats" command

To run the command, click the "Manage Mark Formats" button, which is located in the hidden part of the "Marking" panel of the "3D Security Designer" tab. After that, the "Manage Mark Formats" window will be displayed.


Fig. 2. The "Manage Mark Formats" window

The window control elementss are responsible for the following functionality:

  1. The "Choose Format" drop-down list allows you to select one of the formats previously created in the document to delete. You can delete only formats that are not used for marking existing circuits in the document. If the selected format in the drop-down list is not used in the document, the "Delete" button will be active, otherwise the format cannot be deleted.
  2. The "Format Sample" field gives a string representation of the mark format for the format selected in the drop-down list. The string represents the name and structure of the format.
  3. The "Delete" button allows you to delete the format from the current document, provided that it is not currently used for marking the circuit. In case of successful deletion, the corresponding information message will be displayed.


Fig. 3. Confirmation of deletion

  1. The "Import" button allows you to import mark formats from a text file with the *.arbmf extension. After clicking on the button, the "Import Formats" window will appear, in which you need to select a file to import with the .arbmf extension. Then click on the "Open" button to start the format import process.


Fig. 4. Import of mark formats

If format name matches are found in the imported file and in the document to which the import is being carried out, the corresponding information windows for each match will be displayed.


Fig. 5. Warning when format names match

In case of successful import of mark formats, the corresponding information message will be displayed.


Fig. 6. Information about the successful import of formats

The imported mark formats will be added to the "Choose Formats" drop-down list.


Fig. 7. The "Choose Formats" list with imported mark formats

  1. The "Export" button allows you to export the mark formats from the current document to a text file with the *.armbf extension. After clicking on the button, the "Export Formats" window will appear, in which you need to specify the location on the computer where the file will be saved and specify the file name. Then click on the "Save" button to start the format export process.


Fig. 8. Export of mark formats

In case of successful export of mark formats, the corresponding information message will be displayed.


Fig. 9. Message about successful export of mark formats

  1. The "Cancel" button cancels the "Manage Mark Formats" command and closes the "Manage Mark Formats" window.

If the command fails, a corresponding message will be displayed, and there will be no deletion, import or export of mark formats.


Fig. 10. Failure message
