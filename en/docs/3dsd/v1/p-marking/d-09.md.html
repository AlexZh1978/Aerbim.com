The "Find Element" command

Purpose: finding of the circuit element by mark value. The desired mark value is entered manually.


Fig. 1. Launching the "Find Element" command

To run the command, click the "Find Element" button in the "Actions with Single Elements" group on the "Marking" panel of the 3D Security Designer tab. If no circuits have been created in the current document before, a window will be displayed informing the user about this.


Fig. 2. Warning about the absence of circuits

Otherwise, the "Find Element by Mark Value" window will be displayed.


Fig. 3. The window "Find Element by Mark Value"

The window control elements are responsible for the following functionality:

  1. The text field "Search String" allows you to set a value for searching for element (family) marking in the circuit. The space characters are removed at the beginning and end of the string before searching.
  2. The "Find" button starts the process of searching for elements (families) marking in circuits.
  3. The "Cancel" button cancels the "Find Element" command and closes the "Find Element by Mark Value" window.

After setting the value to search in the "Find Element by Mark Value" window and clicking the "Find" button, a search for mark values for elements (families) in circuits is launched. As a result, the search result will include all the elements in which the mark value string will contain the substring specified for the search in the "Find Element by Mark Value" window.
