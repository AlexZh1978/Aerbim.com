The "Delete Circuit" command

Purpose: deletion of previously applied marks for all elements assigned by the user to one circuit.

When deleting a circuit from the project, the circuit name is deleted, the circuit elements themselves are not deleted from the project. Subsequently, the deleted circuit name can be reused.


Fig. 1. Launching the "Delete Circuit" command

To run the command, click the "Delete Circuit" button in the "Actions with Circuits" group on the "Marking" panel of the "3D Security Designer" tab. If no circuits have been created in the current document before, a window will be displayed informing the user about this.


Fig. 2. Warning about the absence of circuits

Otherwise, the "Delete Circuit and Marks" window will be displayed.


Fig. 3. The "Delete Circuit and Marks" window

The window control elements are responsible for the following functionality:

  1. The "Circuit Name" list allows you to select one of the circuits previously created in the document to delete marks.
  2. The "Information on Existing Circuits" button allows you to open a window reflecting information about marked circuits in the document.
  3. If the "Delete Alternative Marks" checkbox is selected, then when a mark is deleted from the element (family) the value of the alternative mark (if any) will also be deleted. If the visualization remains, then a question symbol is displayed instead of marking.
  4. If the "Delete Mark Visualization" checkbox is selected, then when deleting a mark from an element (family), its visual representation (if any) will also be deleted.
  5. The "Delete Circuit and Mark" button starts the process of deleting marks for the selected circuit.
  6. The "Cancel" button cancels the "Delete Circuit" command and closes the "Delete Circuit and Marks" window.

After setting up the logic of the "Delete Circuit" command in the "Delete Circuit and Marks" window and clicking the "Delete Circuit and Mark" button, a window will appear with a request to confirm the circuit deletion.


Fig. 4. Confirmation of circuit deletion

After confirmation, the process of deleting the mark of the previously selected circuit starts in accordance with the specified settings. At the same time, the "Delete Circuit and Marks" window does not close, but the circuit being deleted disappears from the list of existing circuits and from the current document. You can select and delete circuits in the "Delete Circuit and Marks" window as long as there are available circuits in the list.
