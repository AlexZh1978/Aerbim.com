The "Registration" command

Purpose: registration of the "3D Security Designer" plugin on the local computer for the current version of Autodesk Revit.


Fig. 1. Launching the "Registration" command

To get started and activate the panels included in the "3D Security Designer" plugin (add-in), you must log in to an account that was previously created on the site "www.aerbim.com ". The view of the panels before logging in to the account is shown in Figure 2.


Fig. 2. View of the panels included in the "3D Security Designer" plugin (add-in) before logging in to the account

Login to the account is performed by the "Registration" command. To launch it, click the "Registration" button on the "Registration and Update" panel of the "3D Security Designer" tab. After that, the "Sign in to AerBIM account" window will be displayed (рис. 3).


Fig. 3. The "Sign in to AerBIM account" window

  1. The "Login (e-mail)" field is intended for entering a login. A login is a user's email address. The login must match the username under which the user account was created on the site www.aerbim.com .
  2. The "Password" field is intended for entering a password. The password must match the password that was created for the user account on the site www.aerbim.com .
  3. The link "Create/edit an AerBIM account" navigates to the site www.aerbim.com to create/edit an account.
  4. The "Sign in" button logs in to the account. The button is active only if all text fields are filled in.
  5. The "Sign out" button logs out of the account. The button is active only if you have previously logged in to your account.

After entering the login and password values as part of the "Registration" command in the "Sign in to AerBIM account" window and clicking the "Sign In" button, the activation process of Autodesk Revit panels starts. At the same time:

  1. If the login and password are correct and the license (test period) has not expired, then the panels included in the "3D Security Designer" plugin (add–in) will be activated, i.e. available for operation - Fig. 4, and in the "Sign in to the AerBIM account" window reference information about the validity period of the license (test period) will appear (Fig. 5).


Fig. 4. View of the panels included in the plugin (add-in) "3D Security Designer", after logging in to the account, provided that the data entered correctly and the valid license period (test period)


Fig. 5. The "Sign in to the AerBIM account" window with reference information about the validity period of the license (test period)

  1. If the entered password and/or login are not correct, the activation of the panels will not occur, and the corresponding reference information will appear in the "Sign in to the AerBIM account" window, highlighted in red (Fig. 6).


Fig. 6. The "Sign in to the AerBIM account" window with reference information about an incorrectly entered username and/or password

  1. If the login and password are correct, but the license (test period) has expired, the activation of the panels will not occur, and the corresponding reference information will appear in the "Sign in to the AerBIM account" window, highlighted in red (Fig. 7).


Fig. 7. The "Sign in to the AerBIM account" window with reference information about the expired test period (message highlighted in red)

The following situations and ambiguities are possible when working with the "Registration" command:

  1. If there is no connection to the server after clicking on the "Sign in" button in the "Sign in to AerBIM account" window, the corresponding information message highlighted in red will be displayed (Fig. 8). In this case, the activation of the panels will not occur.


Fig. 8. Informational message in the "Sign in to the AerBIM account" window about the absence of a connection to the server

If the command fails, a corresponding information message will be displayed.
