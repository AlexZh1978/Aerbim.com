Панель «Адаптация»

Назначение: Панель инструментов «Адаптация» предназначена для:

  • Адаптации существующих семейств к работе с надстройкой 3D Security Designer c возможностью адаптации вложенных семейств.

  • Адаптации существующих семейств к работе с шаблоном «ARBM_ШаблонПроекта_СПС-СОУЭ_R2019(20)_v1.1.rte».

  • Скачивания шаблона «ARBM_ШаблонПроекта_СПС-СОУЭ_R2019(20)_v1.1.rte».

  • Скачивания руководства по работе с шаблоном «ARBM_ШаблонПроекта_СПС-СОУЭ_R2019(20)_v1.1.rte».

Терминология

Для однозначного понимания принципов работы и пользовательского интерфейса надстройки необходимо понимать, в каком смысле используются следующие термины:

Адаптация семейств — дополнительная автоматическая параметризация семейств.

Адаптация к надстройке — дополнительная автоматическая параметризация семейств для обеспечения работы с надстройкой 3D Security Designer.

Адаптация к шаблону — дополнительная автоматическая параметризация семейств для обеспечения работы с шаблоном «ARBM_ШаблонПроекта_СПС-СОУЭ_R2019(20)_v1.1.rte».

Вложенное семейство — загружаемое семейство, которое используется внутри другого семейства с возможностью установки зависимостей.

Семейство — группа схожих элементов, которая характеризуется общим набором свойств и связанных с ними графических представлений. Каждое семейство принадлежит одной конкретной категории и получает из категории набор первичных параметров. Остальные параметры задаются пользователем (при необходимости).

Параметризация — процесс присвоения параметров объектам (семействам).

Параметр — свойство семейств Autodesk Revit. Параметр может быть системным (заданным для конкретной категории на программном уровне) или пользовательским (созданным пользователем). Параметры служат для хранения и отображения информации о семействе. Параметры позволяют изменять семейство без необходимости его редактирования в редакторе семейств. Информация может вноситься в параметр в процессе разработки семейства пользователем, либо рассчитываться автоматически на основании данных из модели.

Шаблон проекта — файл с расширением RTE, содержащий ряд настроек параметров проекта и семейств, используемый для создания новых проектов.

Надстройка (плагин) — независимо компилируемый программный модуль, динамически подключаемый к основной программе и предназначенный для расширения и/или использования её возможностей.

Коллизия — логическое, геометрическое или физическое противоречие (либо неоднозначность) в указаниях пользователя, делающее невозможным реализацию этих указаний. Перед реализацией указаний требуется устранение коллизий.

Команды панели "Адаптация"

Панель «Адаптация» предоставляет команды для выполнения следующих действий:

  • Адаптации семейств к плагину.
  • Адаптации семейств к шаблону.

Примечание

Команды из панели «Адаптация» работают только для следующих категорий семейств Revit (в классификации Autodesk):

  • «Пожарная сигнализация».
  • «Охранная сигнализация».
  • «Датчики».
  • «Электрооборудование».
  • «Силовые электроприборы».
  • «Устройства связи».
  • «Осветительные приборы».
  • «Устройства для вызова и оповещения».
  • «Телефонные устройства».
