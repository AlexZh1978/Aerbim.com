Команда «Удаление элементов»

Назначение: удаление элементов из цепи применяется в случае, если нужно исключить элементы из цепи с удалением их маркировки, в том числе альтернативной. Удаленные элементы более не принадлежат цепи, не имеют маркировки, но остаются в проекте. Удаление элементов из цепи может сопровождаться автоматическим обновлением маркировки оставшихся в цепи элементов.


Рис. 1. Запуск команды «Удаление элементов»

Для запуска команды нажмите кнопку «Удаление элементов» в группе «Действия с группой элементов» на панели инструментов «Маркировка» вкладки «3D Security Designer». Если в текущем документе ранее не было создано ни одной цепи, то отобразится окно, информирующее об этом пользователя,

Иначе отобразится окно «Удаление элементов из цепи».


Рис. 3. Окно «Удаление элементов из цепи»

Управляющие элементы окна отвечают за следующий функционал:

  1. Список «Выбор цепи» позволяет выбрать одну из ранее созданных в документе цепей для удаления из нее элементов (семейств).
  2. Поле «Формат маркировки цепи» даёт представление о формате маркировки для выбранной цепи. Поле содержит через двоеточие имя формата и структуру формата.
  3. Если флажок «Изменить маркировку оставшихся элементов» установлен, то после удаления элементов будет автоматически изменяться маркировка (порядок следования элементов) и ее визуальное представление для оставшихся в цепи элементов.
  4. Таблица «Информация о цепи» содержит количественную информацию о выбранной цепи включая поля:
  • «Количество элементов (семейств)» — количество элементов (семейств) в цепи, включая вложенные.
  • «Макс. количество адресов формата» — максимальное количество номеров, предусмотренное форматом маркировки.
  • «Количество незанятых адресов» — количество неиспользованных номеров, которые можно присвоить элементам, не нарушая требования формата маркировки.
  • «Количество адресов за пределами формата» — количество номеров элементов, лежащих за пределами, предусмотренными форматом маркировки.
  1. Кнопка «Удалить элементы из цепи» запускает процесс удаления элементов из выбранной цепи.
  2. Кнопка «Отмена» отменяет команду «Удаление элементов» и закрывает окно «Удаление элементов из цепи».

После настройки логики работы команды «Удаление элементов» в окне «Удаление элементов из цепи» и нажатия кнопки «Удалить элементы из цепи» запускается удаление элементов из выбранной цепи. Если требуется удалить элементы, находящиеся на разных видах, то перед запуском этой команды необходимо предварительно открыть данные виды, чтобы по ним можно было переключаться в процессе работы команды. При этом все элементы (если они присутствуют на виде), входящие в цепь, подсветятся зеленым цветом на текущем виде. В строке состояния появится запрос «Выделите элементы, которые будут удалены из цепи». После выбора удаляемых элементов, нажмите кнопку «Готово» на панели параметров, чтобы завершить выбор. В результате из цепи будут удалены выбранные элементы с учетом сделанных ранее настроек в окне «Удаление элементов из цепи», а оставшиеся элементы цепи будут подсвечены. Если будут удалены все элементы цепи, то цепь полностью удалится из текущего проекта.

ВАЖНО: для ускорения выполнения команды - визуализация маркировки у элементов после изменения цепи, будет отображена только на тех видах, где она была до выполнения команды. Это значит, что если визуализации у элементов изменяемой цепи на каких-либо видах не было, то ее не будет на этих видах и после выполнения команды, не смотря на значение шага визуализации.
