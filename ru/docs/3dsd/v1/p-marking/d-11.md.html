Команда «Управление форматами»

Назначение: дать возможность пользователю удалять существующие форматы, экспортировать форматы в текстовый файл, а также импортировать их из текстового файла.


Рис. 1. Вызов команды «Управление форматами»

Для запуска команды нажмите кнопку «Управление форматами», которая расположена в разворачиваемой части на панели инструментов «Маркировка» вкладки «3D Security Designer». После этого отобразится окно «Управление форматами».


Рис. 2. Окно «Управление форматами»

Управляющие элементы окна отвечают за следующий функционал:

  1. Выпадающий список «Форматы» позволяет выбрать один из ранее созданных в документе форматов для удаления. Удалить можно только форматы, которые не использованы при маркировке существующих в документе цепей. Если выбранный формат в выпадающем списке не использован в документе, то кнопка «Удалить» будет активна, в противном случае формат удалить не удастся.
  2. Поле «Структура формата» даёт строковое представление формата маркировки для выбранного в выпадающем списке формата. Строка представляет собой имя и структуру формата.
  3. Кнопка «Удалить» позволяет удалить формат из текущего документа, при условии, что он не используется в настоящий момент для маркировки цепи. В случае успешного удаления отобразится соответствующее информационное сообщение.


Рис. 3. Подтверждение удаления

  1. Кнопка «Импорт» позволяет импортировать форматы маркировки из текстового файла с расширением *.arbmf. После нажатия на кнопку, отобразиться окно «Импорт форматов», в котором требуется выбрать файл для импорта с расширением arbmf. Затем нажать на кнопку «Открыть» для запуска процесса импорта форматов.


Рис. 4. Импорт форматов маркировки

Если в импортируемом файле и в документе, в который осуществляется импорт, будут обнаружены совпадения имен форматов, то будут отображаться соответствующие информационные окна по каждому совпадению.


Рис. 5. Предупреждение при совпадении имён форматов

В случае успешного импорта форматов маркировки будет выведено соответствующее информационное сообщение.


Рис. 6. Информация об успешном импорте форматов

Импортированные форматы маркировки добавятся в выпадающий список «Форматы».


Рис. 7. Список «Форматы» с импортированными форматами

  1. Кнопка «Экспорт» позволяет экспортировать форматы маркировки из текущего документа в текстовый файл с расширением *.armbf. После нажатия на кнопку, отобразиться окно «Экспорт форматов», в котором требуется указать место на компьютере, где будет сохранен файл и задать имя файла. Затем нажать на кнопку «Сохранить» для запуска процесса экспорта форматов.


Рис. 8. Экспорт форматов маркировки

В случае успешного экспорта форматов маркировки будет выведено соответствующее информационное сообщение.


Рис. 9. Сообщение об успешном экспорте форматов маркировки

  1. Кнопка «Отмена» отменяет команду «Управление форматами» и закрывает окно «Управление форматами».

В случае сбоя команды будет выведено соответствующее сообщение, при этом удаления, импорта или экспорта форматов маркировки не произойдет.
