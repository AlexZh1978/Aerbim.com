Команда «Информация о маркировке»

Назначение: дать возможность пользователю быстро получить информацию о маркировке для выбранного элемента. Выводимая информация: имя цепи, формат маркировки, значение маркировки, альтернативное значение маркировки.


Рис. 1. Запуск команды «Информация о маркировке»

Для запуска команды нажмите кнопку «Информация о маркировки» в группе «Действия с элементом» на панели инструментов «Маркировка» вкладки «3D Security Designer». Если в текущем документе ранее не было создано ни одной цепи, то отобразится окно, информирующее об этом пользователя.


Рис. 2. Предупреждение об отсутствии цепей

Иначе в строке состояния появится запрос «Выделите элемент, для которого необходимо получить информацию о его маркировке». После выделения элемента (семейства) в цепи отобразится окно «Информация о маркировке элемента».


Рис. 3. Окно «Информация о маркировке элемента»

Управляющие элементы окна отвечают за следующий функционал:

  1. Список пар «Параметр» - «Значение» содержит значения, участвующие в формировании маркировки указанного элемента (семейства). К таким параметрам относятся: «Имя цепи», «Формат маркировки», «Маркировка», «Альтернативная маркировка».
  2. Кнопка «ОК» завершает команду «Информация о маркировке» и закрывает окно «Информация о маркировке элемента».

В случае сбоя команды будет выведено соответствующее сообщение, при этом информация о маркировке элемента отображена не будет.
