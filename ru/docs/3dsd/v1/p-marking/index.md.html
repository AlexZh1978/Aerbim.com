Панель «Маркировка»

Предназначение: панель «Маркировка» предназначена для автоматической маркировки элементов, логически объединённых в цепи. Панель позволяет работать с такими элементами на уровне цепей, групп элементов и отдельных элементов.

Маркирование элементов, т.е. соотнесение каждому элементу строковой величины, идентифицирующей данный элемент в проекте, является одной из самых рутинных операций. Маркирование осложняется тем, что в процессе проектирования элементы цепей могут быть как добавлены, так и удалены. Это часто происходит с извещателями пожарной и охранной сигнализаций.

Методология BIM предполагает автоматизацию подобных рутинных процессов для избежания коллизий и ускорения проектирования, что и достигается с помощью данной панели.

Терминология

Для однозначного понимания принципов работы и пользовательского интерфейса надстройки необходимо понимать, в каком смысле используются следующие термины:

Маркировка (сущ.) – строковая величина с определяемой пользователем структурой, идентифицирующая экземпляр семейства Revit в составе проекта, а также идентифицирующая соответствующее семейству изделие в проектной / монтажной / наладочной и т.п. документации.

Маркировать (гл.) – наносить маркировку.

Маркирование (сущ.) – нанесение маркировки.

Нанесение маркировки – процесс ручного или автоматического соотнесения конкретным семействам Revit конкретных маркировок.

Формат маркировки – определяемая пользователем структура маркировки. Как правило, маркировка содержит в некотором порядке буквенно-цифровые обозначения, порядковые номера, символы-разделители и т.п.

Визуализация маркировки – передача визуальными средствами информации о маркировке семейства в проектной среде, а также о маркировке соответствующего семейству изделия в проектной / монтажной / наладочной и т.п. документации.

Цепь – подмножество экземпляров семейств Revit в составе проекта, выделенное пользователем, и имеющее указанный пользователем порядок следования. Порядок следования семейств может быть использован как для нанесения маркировки, так и для дальнейшего проектирования слаботочных электрических систем, в т.ч. автоматизированного.

Коллизия – логическое, геометрическое или физическое противоречие (либо неоднозначность) в указаниях пользователя, делающее невозможным реализацию этих указаний. Перед реализацией указаний требуется устранение коллизий.

Команды панели «Маркировка»

Панель «Маркировка» предоставляет команды для выполнения следующих действий:

  • Нанесения маркировки цепи в соответствии с предварительно выбранным форматом маркировки. Маркирование учитывает наличие вложенных семейств и множественность адресов для одного устройства (в случае BIM-проектирования адресных систем безопасности).
  • Изменения существующей маркировки в цепи.
  • Удаления маркировки цепи.
  • Получения информации о маркировки цепи.
  • Добавления промаркированных элементов к цепи.
  • Удаления промаркированных элементов из цепи.
  • Задания альтернативной маркировки для элементов цепи.
  • Получения информации о маркировки элемента цепи.
  • Поиска элементов цепи по значению маркировки.
  • Создания форматов маркировки.
  • Управления форматами маркировки.
  • Управления визуализацией маркировки.

Примечание

Команды из панели «Маркировка» работают только для следующих категорий семейств Revit (в классификации Autodesk):

  • «Пожарная сигнализация».
  • «Охранная сигнализация».
  • «Датчики».
  • «Электрооборудование».
  • «Силовые электроприборы».
  • «Устройства связи».
  • «Осветительные приборы».
  • «Устройства для вызова и оповещения».
  • «Телефонные устройства».

Маркируемые семейства должны быть адаптированы к плагину. Процесс адаптации заключается в создании специальных параметров семейства. Процесс адаптации может быть автоматизирован с помощью панели «Адаптация» в составе плагина.
