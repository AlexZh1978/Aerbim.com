Команда «Удаление цепи»

Назначение: удаление ранее нанесенной маркировку для всех элементов, отнесенных пользователем в одну цепь.

При удалении цепи из проекта удаляется имя цепи, сами элементы цепи из проекта не удаляются. Впоследствии удаленное имя цепи можно использовать повторно.


Рис. 1. Запуск команды «Удаление цепи»

Для запуска команды нажмите кнопку «Удаление цепи» в группе «Действия с цепями» на панели инструментов «Маркировка» вкладки «3D Security Designer». Если в текущем документе ранее не было создано ни одной цепи, то отобразится окно, информирующее об этом пользователя.


Рис. 2. Предупреждение об отсутствии цепей

Иначе отобразится окно «Удаление маркировки цепи».


Рис. 3. Окно «Удаление маркировки цепи»

Управляющие элементы окна отвечают за следующий функционал:

  1. Список «Имя цепи» позволяет выбрать одну из ранее созданных в документе цепей для удаления маркировки.
  2. Кнопка «Информация о маркированных цепях» - позволяет вызвать окно, отражающее информацию о маркированных цепях в документе.
  3. Если флажок «Удалить альтернативную маркировку» установлен, то при удалении маркировки из элемента (семейства) удалится и значение альтернативной маркировки (в случае ее наличия). Если визуализация остаётся, то вместо маркировки отображается символ вопроса.
  4. Если флажок «Удалить визуализацию маркировки» установлен, то при удалении маркировки из элемента (семейства) удалится и ее визуальное представление (в случае его наличия).
  5. Кнопка «Удалить маркировку» запускает процесс удаления маркировки для выбранной цепи.
  6. Кнопка «Отмена» отменяет команду «Удаление цепи» и закрывает окно «Удаление маркировки цепи».

После настройки логики работы команды «Удаление цепи» в окне «Удаление маркировки цепи» и нажатия кнопки «Удалить маркировку» отобразится окно с запросом на подтверждение удаления цепи.


Рис. 4. Подтверждение удаления цепи

После подтверждения запускается процесс удаления маркировки выбранной ранее цепи в соответствии с заданными настройками. При этом окно «Удаление маркировки цепи» не закрывается, но из списка существующих цепей и из текущего документа удаляемая цепь исчезает. Выбирать и удалять цепи в окне «Удаление маркировки цепи» можно до тех пор, пока в списке есть доступные цепи.
