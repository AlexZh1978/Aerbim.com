Команда «Поиск элемента»

Назначение: быстрое нахождение элемента цепи по значению маркировки. Искомое значение маркировки вводится вручную.


Рис. 1. Запуск команды «Поиск элемента»

Для запуска команды нажмите кнопку «Поиск элемента» в группе «Действия с элементом» на панели инструментов «Маркировка» вкладки «3D Security Designer». Если в текущем документе ранее не было создано ни одной цепи, то отобразится окно, информирующее об этом пользователя.


Рис. 2. Предупреждение об отсутствии цепей

Иначе отобразится окно «Поиск элементов по маркировке».


Рис. 3. Окно «Поиск элементов по маркировке»

Управляющие элементы окна отвечают за следующий функционал:

  1. Текстовое поле «Введите строку для поиска» позволяет задать значение для поиска маркировки элемента (семейства) в цепи. Символы пробелов удаляются в начале и конце строки перед поиском.
  2. Кнопка «Найти» запускает процесса поиска маркировки у элементов (семейств) в цепях.
  3. Кнопка «Отмена» отменяет команду «Поиск элемента» и закрывает окно «Поиск элементов по маркировке».

После задания значения для поиска в окне «Поиск элементов по маркировке» и нажатия кнопки «Найти» запускается поиск значений маркировки у элементов (семейств) в цепях. В итоге в результат поиска попадут все элементы, у которых строка со значением маркировки будет содержать подстроку, указанную для поиска в окне «Поиск элементов по маркировке».
