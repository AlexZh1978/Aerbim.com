Команда «Обновление»

Назначение: обновление плагина «3D Security Designer».


Рис. 1. Запуск команды «Обновление»

Команда активна только в случае наличия обновления у разработчика плагина. Если обновления отсутствуют, то кнопка запуска команды «Обновление» будет не активна (рис. 2).


Рис. 2. Вид кнопки «Обновление», когда она не активна

Для запуска команды нажмите кнопку «Обновление» на панели инструментов «Регистрация и обновление» вкладки «3D Security Designer». После этого будет запущен интернет-браузер и осуществлен переход на страницу сайта разработчика плагина, с которой можно будет скачать обновление в виде инсталляционного пакета (файл с расширением «msi»).

Для установки обновления необходимо закрыть Autodesk Revit. После этого запустить процесс инсталляции (двойной клик «мыши» по скачанному файлу с обновлением) и следовать инструкциям инсталлятора.
