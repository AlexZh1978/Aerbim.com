Панель «Регистрация и обновление»

Предназначение: панель «Регистрация и обновление» предназначена для:

  • Входа/выхода в учетную запись для активации/деактивации панелей инструментов, входящих в состав плагина (надстройки) «3D Security Designer».

  • Получения и установки обновлений.

  • Получения информации о надстройки (плагине) «3D Security Designer».


Рис. 1. Вид панели «Регистрация и обновление»

Терминология

Для однозначного понимания принципов работы и пользовательского интерфейса надстройки необходимо понимать, в каком смысле используются следующие термины.

Учётная запись — хранимая в компьютерной системе совокупность данных о пользователе, необходимая для его опознавания (аутентификации) и предоставления доступа к его личным данным и настройкам.

Регистрация плагина “3D Security Designer” (программного обеспечения) — обозначает владельца зарегистрированной программы и служит инструментом защиты его интересов и авторских прав автора.

Обновление плагина “3D Security Designer” (программного обеспечения) — представляет собой набор изменений для программного обеспечения, исправления или улучшения.

Категория семейства — группа элементов, используемых для моделирования объекта строительства: пожарная сигнализация, устройства для вызова и оповещения, окна, двери, стены, перекрытия и др.

Коллизия — логическое, геометрическое или физическое противоречие (либо неоднозначность) в указаниях пользователя, делающее невозможным реализацию этих указаний. Перед реализацией указаний требуется устранение коллизий.

Команды панели «Регистрация и обновление»

Панель «Регистрация и обновление» предоставляет команды для выполнения следующих действий:

  • Входа/выхода в учетную запись.
  • Обновления надстройки (плагина).
  • Получения информации о надстройки (плагине) «3D Security Designer».
