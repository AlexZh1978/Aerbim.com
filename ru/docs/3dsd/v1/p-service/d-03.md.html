Команда «О программе»

Назначение: отображение информации о программе «3D Security Designer» версии 1.0.


Рис. 1. Запуск команды «О программе»

Для запуска команды нажмите кнопку «О программе» на панели инструментов «Регистрация и обновление» вкладки «3D Security Designer». После этого отобразится окно «О программе» (рис. 2), которое содержит информацию о названии программы, ее версии, разработчике и технической поддержке.
