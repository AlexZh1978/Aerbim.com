Оплата

На данный момент доступна версия надстройки (плагина) 3D Security Designer для бесплатного тестирования, которую можно получить после обязательной регистрации в Личном кабинете на срок до 45 календарных дней, с возможностью продления срока тестирования по запросу.

После окончания срока тестирования возможен переход на платное использование с предварительным уведомлением пользователя. Информация об условиях платного использования будет предоставлена в уведомлении.

Тестовый период начинается с момента регистрации пользователя в Личном кабинете.
