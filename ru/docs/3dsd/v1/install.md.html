Установка

Программное обеспечение 3D Security Designer (далее плагин) предназначено для работы в среде Autodesk® Revit ® 2019 и Autodesk® Revit ® 2020.

Плагин распространяется в форме дистрибутивов Aerbim.3DSD.v1.r19.ru.msi - для Autodesk® Revit ® 2019 и Aerbim.3DSD.v1.r20.ru.msi - для Autodesk® Revit ® 2020 (Aerbim.3DSD.v1.r19.en.msi и/или Aerbim.3DSD.v1.r20.en.msi для англоязычных локализаций), которые содержат набор специальных файлов, обеспечивающих работу плагина.

ВАЖНО: для инсталляции и успешной работы плагина на целевой компьютер необходимо предварительно установить программу Autodesk® Revit® 2019(2020). До начала запуска установщика плагина требуется закрыть все работающие экземпляры программы Autodesk® Revit® 2019(2020) на целевом компьютере.

Установку плагина должен производить пользователь, обладающий на данном компьютере правами администратора операционной системы. Для установки запустите файл Aerbim.3DSD.v1.ru.msi (Aerbim.3DSD.v1.en.msi для англоязычной локализации). Компоненты плагина устанавливается по следующему пути: c:\Program Files\AerBIM\3D Security Designer for Revit 2019(2020).

Программа установки самостоятельно определит наличие на компьютере программы Autodesk® Revit® 2019(2020) и, в случае ее отсутствия, сообщит об этом (рис. 1), при этом работа установщика будет прервана (рис. 2).
