Надстройка (плагин) 3D Security Designer

Назначение: надстройка (плагин) 3D Security Designer используется для автоматизации рутинных операций при проектировании систем охранной и пожарной безопасности различных производителей в среде Autodesk® Revit®.

Функциональные возможности

Надстройка (плагин) 3D Security Designer предназначена для работы в среде Autodesk® Revit®, разработана с учётом BIM-методологии.

Надстройка (плагин) 3D Security Designer является мультивендорной, т.е. позволяет использовать при проектировании оборудование различных производителей.

Надстройка (плагин) 3D Security Designer содержит несколько панелей инструментов, имеющих отношение к разным аспектам проектирования систем охранной и пожарной безопасности:

  • Панель «Пожарная сигнализация» используется для быстрого поиска, анализа, выбора и вставки в проект семейств Revit из онлайн-хранилища цифровых моделей оборудования различных производителей для систем пожарной сигнализации.
  • Панель «Оповещение о пожаре» используется для быстрого поиска, анализа, выбора и вставки в проект семейств Revit из онлайн-хранилища цифровых моделей оборудования различных производителей для систем оповещения о пожаре.
  • Панель «Маркировка» используется для организации семейств Revit в цепи и автоматического маркирования элементов цепей.
  • Панель «Адаптация» используется для адаптации семейств сторонних разработчиков к работе с надстройкой (плагином) 3D Security Designer и шаблонами AerBIM.
  • Панель «Регистрация и обновление» используется для регистрации и получения обновлений

Особенности работы с интерфейсом

При появлении различных информационных сообщений (рис. 7) или при выполнении команд в процессе работы надстройки (плагина) 3D Security Designer, возможна краткосрочная блокировка (рис. 8) ленты Autodesk® Revit®. Это связано с особенностями функционирования интерфейса самого Revit®. В этом случае лента будет разблокирована в течении нескольких секунд.

Категории семейств Revit

Надстройка (плагин) предназначена для работы только со следующими категориями семейств Revit® (в классификации Autodesk®):

  • «Пожарная сигнализация».
  • «Охранная сигнализация».
  • «Датчики».
  • «Электрооборудование».
  • «Силовые электроприборы».
  • «Устройства связи».
  • «Осветительные приборы».
  • «Устройства для вызова и оповещения».
  • «Телефонные устройства».
