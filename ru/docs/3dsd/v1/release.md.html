Версия 1.0

Первая версия надстройки (плагина) 3D Security Designer предназначена для автоматизации процесса проектирования систем пожарной сигнализации и оповещения о пожаре в среде в среде Autodesk® Revit® и имеет следующие особенности:

  • Мультивендорность.
  • Интуитивно понятный интерфейс.
  • Полная информационная поддержка по описанию работы с плагином в виде справочной системы.
  • Возможность получения информации из ссылочных файлов для реализации функций автоматизации.
  • Модульная архитектура плагина позволяет быстро наращивать его функциональность и адаптировать его к изменению требований пользователей.
  • Динамично расширяемая база данных адаптированных семейств с возможностью дополнительной адаптации под любые требования заказчика.
  • Возможность быстрой адаптации семейств сторонних разработчиков для их полноценного использования в проектной среде, разработанной компанией AerBIM.
  • Локализация справочной системы и интерфейсных элементов плагина на русском и английском языках.
